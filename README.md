## 직무 탐색을 위한 플랫폼 서비스

```프로젝트 기간: 2023.01.04 ~```

>  지역 기반으로 취준생들과 경력 있는 전문가들 간의 인적 네트워크를 중개하는 플랫폼 서비스

-  **현재 연동된 기능**
    - 검색 조건(지역 범위, 전문 분야, 디렉터의 스케줄 유무, 검색어)을 통한 디렉터 목록 조회가 가능합니다.
    - 사용자의 서비스 접속 위치를 통해 지역 인증이 가능합니다.
    - 그 외 로그인, 로그아웃, 회원가입 연동.
